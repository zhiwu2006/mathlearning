# 新增数学题目说明

## 题目概述

本次新增了7道高质量的数学应用题，涵盖了多个数学领域和难度级别，适合小学高年级到初中学生练习。

## 新增题目列表

### 1. 贺年卡利润计算题
- **题目标识**: `profit-calculation-specific-001`
- **难度等级**: 中等 (M)
- **数学领域**: 利润计算、约束条件、整数解
- **知识点**: 方程建立、条件分析、单价总价关系
- **题目内容**: 商店以每张0.21元的价格进了一批贺年卡，共卖14.57元。若每张的售价相同，且不超过买入价格的两倍，则商店赚了多少元？

### 2. 植树人数优化题
- **题目标识**: `tree-planting-specific-001`
- **难度等级**: 中等 (M)
- **数学领域**: 二元一次方程、最优化
- **知识点**: 整数解分析、极值求解、约束条件
- **题目内容**: 两个班植树，一班每人植3棵，二班每人植5棵，共植树115棵。两班人数之和最多为多少？

### 3. 笔的价格优化题
- **题目标识**: `price-optimization-specific-001`
- **难度等级**: 困难 (H)
- **数学领域**: 价格与销量关系、一元二次方程
- **知识点**: 变量设定、方程建立、公式应用
- **题目内容**: 某商店第一天卖出一些笔，第二天每支笔降价1元后多卖出100支，第三天每支笔比第一天涨价3元后比第一天少卖出200支。如果这三天每天卖得的钱相同，那么第一天每支笔售价是多少元？

### 4. 流水行船速度题
- **题目标识**: `boat-speed-specific-001`
- **难度等级**: 困难 (H)
- **数学领域**: 相对速度、流水问题
- **知识点**: 速度分析、方程建立、物理建模
- **题目内容**: 一条河上有A，B两个码头，A在上游，B在下游。甲、乙两人分别从A，B同时出发，划船相向而行，4小时后相遇。如果甲、乙两人分别从A，B同时出发划船同向而行，乙16小时后追上甲。已知甲在静水中划船的速度为每小时6千米，则乙在静水中划船每小时行驶多少千米？

### 5. 数论同余题
- **题目标识**: `number-theory-specific-001`
- **难度等级**: 中等 (M)
- **数学领域**: 同余定理、最小公倍数
- **知识点**: 模运算、同余分析、枚举验证
- **题目内容**: 某个两位数是2的倍数，加1是3的倍数，加2是4的倍数，加3是5的倍数，那么这个两位数是多少？

### 6. 汉字代数题
- **题目标识**: `chinese-character-specific-001`
- **难度等级**: 困难 (H)
- **数学领域**: 汉字代数、约束条件、优化问题
- **知识点**: 变量映射、约束分析、极值求解
- **题目内容**: 在三个词语'尽心尽力'、'力可拔山'和'山穷水尽'中，每个汉字代表1至8之间的数字，相同的汉字代表相同的数字，不同的汉字代表不同的数字。如果每个词语的汉字所代表的数字之和都是19，且'尽'>'山'>'力'，则'水'最大等于多少？

### 7. 工作效率题
- **题目标识**: `work-rate-specific-001`
- **难度等级**: 中等 (M)
- **数学领域**: 工作效率、时间优化
- **知识点**: 方程建立、变量关系、实际问题建模
- **题目内容**: 有一批作业，王老师原计划每小时批改6本。批改了2小时后，他决定每小时批改8本，结果提前3小时批改完。那么这批作业有多少本？

## 教学价值

### 思维能力培养
1. **逻辑推理能力**: 通过条件分析和约束求解，培养学生的逻辑思维
2. **数学建模能力**: 将实际问题转化为数学方程的能力
3. **优化思想**: 在多个约束条件下寻找最优解
4. **抽象思维**: 用符号和变量表示具体问题

### 数学技能提升
1. **方程求解**: 一元一次、二元一次方程的建立和求解
2. **同余理论**: 数论基础知识的应用
3. **物理建模**: 速度、时间、距离关系的应用
4. **约束优化**: 在实际限制条件下的决策分析

### 应用能力训练
1. **实际应用**: 将数学知识应用到生活和实际问题中
2. **多角度思考**: 从不同角度分析和解决问题
3. **验证能力**: 检验答案的合理性和正确性
4. **创新思维**: 寻找不同的解题方法

## 使用建议

### 教师指导
1. 可以根据学生水平选择不同难度的题目
2. 建议先从较简单的题目开始，逐步增加难度
3. 鼓励学生多种解法，比较不同方法的优劣
4. 注重解题过程的指导，而非仅仅关注答案

### 学生学习
1. 仔细阅读题目，理解所有条件和约束
2. 尝试用不同的方法解决问题
3. 验证答案的正确性和合理性
4. 总结解题经验，形成自己的解题策略
